navigationService
service in module umbraco.services

Description

Service to handle the main application navigation. Responsible for invoking the tree Section navigation and search, and maintain their state for the entire application lifetime

  • syncTree(args)

    Syncs a tree with a given path, returns a promise The path format is: ["itemId","itemId"], and so on so to sync to a specific document type node do: 

    navigationService.syncTree({tree: 'content', path: ["-1","123d"], forceReload: true});
    Parameters
    ParamTypeDetails
    argsObject

    arguments passed to the function

    args.treeString

    the tree alias to sync to

    args.pathArray

    the path to sync the tree to

    args.forceReloadBoolean

    optional, specifies whether to force reload the node data from the server even if it already exists in the tree currently

    args.activateBoolean

    optional, specifies whether to set the synced node to be the active node, this will default to true if not specified
