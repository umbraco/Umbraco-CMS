Audio et vidéo

Cette condition est destinée à agir comme un rappel amical. Les vidéos recevront un avertissement pour s'assurer que les sous-titres sont disponibles. L'audio recevra un avertissement pour s'assurer que le podcast/audio a une transcription. Cette condition ne teste pas si les sous-titres ou les transcriptions sont présents. Personnalisez cette condition en ajoutant vos propres sources de diffusion multimédia.

Widget Twitter

Cette condition vise à encourager les utilisateurs à optimiser le widget Twitter par défaut pour l'accessibilité et la convivialité. Cette condition peut être utile dans les grandes organisations (comme une université ou un collège) où il existe de nombreux canaux de médias sociaux. Cette condition n'est pas basée sur WCAG 2.0, mais est fortement recommandée.

Detecting URL changes

This is not a ruleset. This is a demo of a nifty feature to find errors or warnings after detecting a URL change on the same page. For example, single page applications (SPA). A non-descript link will be appended to the page 1 second after the page is fully loaded. It will not recieve an error flag initially.

To test: Wait for the "click here" link to load. Then click on any of the sidebar navigation links. It will then get flagged as an error.
