Аудіо та відео

Ця умова покликана діяти як дружнє нагадування. Відео отримають попередження, щоб забезпечити доступність субтитрів. Аудіо отримає попередження, щоб переконатися, що подкаст/аудіо має розшифровку. Ця умова не перевіряє наявність субтитрів чи стенограм. Налаштуйте цю умову, додавши власні джерела потокового медіа.

Виявлення змін URL

Це не набір правил. Це демонстрація чудової функції для пошуку помилок або попереджень після виявлення зміни URL-адреси на тій самій сторінці. Наприклад, односторінкові програми (SPA). Посилання без опису буде додано до сторінки через 1 секунду після повного завантаження сторінки. Спочатку він не отримає позначку про помилку.

Щоб перевірити: дочекайтеся завантаження посилання «натисніть тут». Потім натисніть будь-яке з навігаційних посилань бічної панелі. Тоді це буде позначено як помилка.
